# PPT - Pedra, Papel e Tesoura
#### Um jogo interativo de pedra, papel e tesoura desenvolvido em JavaScript com foco em praticar conceitos fundamentais de estilização e lógica.

## Funcionalidades
- Escolha entre Pedra, Papel ou Tesoura
- Sistema automático que gera a jogada do computador
- Interface simples e responsiva
- Exibição clara do resultado da rodada (vitória, derrota ou empate)
- Contador de pontuação para jogador e computador

## Tecnologias
- HTML5: estrutura da aplicação
- CSS3: estilização e organização dos elementos
- JavaScript: lógica do jogo e interação com o usuário

## Como Executar

1. Clone esse repositório:
```bash
git clone https://github.com/mariaeduardadepascoa/PPT
```

2. Abra o index.html no navegador.
